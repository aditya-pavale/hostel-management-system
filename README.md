# Hostel Management System

This project is a web-based Hostel Management System developed as an academic
project to manage student accommodation and administrative operations efficiently.

## Overview
The system provides separate access for administrators and students, allowing
for streamlined room allocation, student registration, and record management.
It focuses on database design, role-based access, and structured workflows.

## Features
- Student registration and login
- Admin authentication and dashboard
- Room allocation and fee management
- Course and student information management
- Login activity tracking

## Technologies Used
- PHP
- HTML and CSS
- MySQL
- XAMPP (Apache & MySQL)

## Database Design
The system uses a relational database with tables for:
- Admin and admin logs
- Student registration and login logs
- Rooms and courses
- State and location data
